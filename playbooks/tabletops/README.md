# Tabletop Exercise Kits

Scenario-based tabletop exercise kits for public-sector leadership, IT, security, communications, and business owners.

## Available tabletop kits

### KEV Exploitation
- **Path:** `playbooks/tabletops/kev-exploitation/`
- **Expected time:** 60–90 minutes
- **Use when:** you want to exercise emergency patching, downtime approval, compensating controls, vendor friction, and leadership decision-making around a known exploited vulnerability.
- **Best for:** IT leadership, security, infrastructure owners, executives, communications.
- **Includes:** Spectral Raccoon / **APT-1337** fictional adversary materials as part of the standard scenario package.

### Ransomware
- **Path:** `playbooks/tabletops/ransomware/`
- **Expected time:** 60–90 minutes
- **Use when:** you want to exercise containment, restoration priorities, communications, continuity, and recovery decisions during a ransomware event.
- **Best for:** IT leadership, security, backup owners, executives, communications, operations.

## Using these kits
Each tabletop is designed to stand on its own. Use the scenario folder that matches the problem you want to practice, review the facilitator guide, tailor the exercise to your environment, and capture follow-up actions in the hotwash.

New tabletop kits can be added over time as separate scenario folders without changing the rest of the collection.
