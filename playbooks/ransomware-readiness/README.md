# Ransomware Readiness (Cities/Counties)

A lightweight readiness package for public-sector environments.

## Contents
- Readiness checklist (prevent/detect/respond)
- Tabletop exercise outline
- Communications templates

## Quick checklist (minimum viable)
- MFA everywhere (VPN, email, admin)
- Immutable/offline backups + restore tests
- EDR coverage and alerting
- Central logging (VPN, firewall, DNS, EDR, cloud audit)
- Admin segmentation (separate admin workstations/accounts)
