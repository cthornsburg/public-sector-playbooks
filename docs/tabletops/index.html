Tabletop Exercise Kits

Scenario-based exercises for public-sector teams

Use these tabletop kits to practice coordination, communications, decision-making, and follow-through before a real incident forces the lesson.
Main start page GitHub source

How to use these exercises

Each exercise is designed to stand on its own. Pick the scenario that fits your goal, review the facilitator guide, bring the right people into the room, and capture follow-up actions in the hotwash.

  • Expected time: most exercises are built for 60–90 minutes
  • Best audience: leadership, IT, security, communications, and key service owners
  • Goal: better decisions, cleaner escalation, and fewer surprises during real events
