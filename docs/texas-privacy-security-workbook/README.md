# Texas Privacy & Security Workbook (Cities/Counties + Private Orgs) (Unofficial)

This is a **community, unofficial** workbook covering practical privacy/security obligations and breach readiness for **Texas** organizations, with emphasis on **cities/counties** and also including items applicable to many **private organizations**.

## Important notes

- **Guide, not legal advice.** This workbook is plain-language and operational. It is **not** legal advice.
- **Scope varies.** Texas obligations differ based on entity type, data types, and contractual/regulatory regimes.
- **No sensitive data.** Do not paste regulated or sensitive information.
- **Local-only storage.** “Save Session” uses `localStorage` and is **not encrypted**.
