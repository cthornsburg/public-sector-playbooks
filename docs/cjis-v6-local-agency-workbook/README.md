# CJIS v6.0 Local Agency Self‑Assessment Workbook (Unofficial)

This is a **community, unofficial** workbook designed to help local agencies prepare for a CJIS Security Policy v6.0 audit.

## Important notes

- **Not an official FBI/CSA form.** This tool is not endorsed by the FBI, CJIS Division, or any state CJIS Systems Agency (CSA).
- **Do not enter CJI.** Do not paste Criminal Justice Information (CJI) or sensitive case data into this workbook.
- **Local-only storage.** The **Save Session** button uses your browser’s `localStorage` on *this device* and is **not encrypted**. Do not use on shared/public computers.
- **Authority.** Your state CSA guidance is authoritative for interpretations, scoping, and audit requirements.

## Official reference

- FBI (official): CJIS Security Policy v6.0 landing page (PDF view)
  https://le.fbi.gov/file-repository/cjis_security_policy_v6-0_20241227.pdf/view

## How to use

1. Open the hosted tool page (GitHub Pages) and complete the workbook.
2. Use **Export JSON** to save your progress to a file.
3. Use **Import JSON** to restore a saved session.
4. Use **Print / PDF Export** for offline review.
