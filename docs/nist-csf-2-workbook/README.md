# NIST CSF 2.0 Workbook (Unofficial)

This is a **community, unofficial** self-assessment workbook aligned to the **NIST Cybersecurity Framework (CSF) 2.0**.

## Important notes

- **Guide, not policy.** This workbook is a practical guide for cities/counties and similar organizations. It is **not** a complete security program or policy set.
- **No sensitive data.** Do not paste secrets or regulated/sensitive data into this tool.
- **Local-only storage.** “Save Session” uses your browser’s `localStorage` on *this device* and is **not encrypted**.
- **Authority.** Your organization’s legal counsel, regulators, auditors, and governing policies are authoritative.
