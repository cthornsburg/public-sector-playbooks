# AI Governance Kit

**Problem this solves:** You need a practical starting point for AI acceptable-use, governance, and basic oversight in a city/county environment.

**Best for:** executives, supervisors, policy owners, legal/procurement collaborators, and IT/security staff.

**Prefer the web version?** Start here: https://cthornsburg.github.io/public-sector-playbooks/ai-governance-starter-pack/

## If you only have 30–60 minutes
1. Inventory current AI use cases.
2. Use the cheat sheet as an interim acceptable-use memo.
3. Start the use-case register.
4. Route the full policy through governance/legal review.

## Good-enough outcome
You should end with:
- a visible list of current AI uses
- interim guardrails for staff
- a draft path toward formal policy approval
