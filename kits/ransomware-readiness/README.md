# Ransomware Readiness Kit

**Problem this solves:** You need a practical ransomware-preparedness starting point for a city, county, or other small public-sector environment.

**Best for:** IT leadership, operations teams, supervisors, and executives who need a fast readiness baseline.

**Prefer the web version?** Start here: https://cthornsburg.github.io/public-sector-playbooks/ransomware-readiness/

## If you only have 30 minutes
1. Confirm MFA coverage for email, VPN, and admin access.
2. Confirm backups exist and restores are actually tested.
3. Verify EDR coverage and alert routing.
4. Confirm minimum logging on key systems.
5. Schedule a tabletop using the ransomware template.

## Good-enough outcome
You should leave with:
- a short readiness gap list
- 3 to 5 priority actions
- ownership for backups, logging, and comms
- a scheduled tabletop or leadership review

## Notes
- This kit is designed to help non-specialists start quickly.
- For leadership use, the hosted page is the easiest first stop.
