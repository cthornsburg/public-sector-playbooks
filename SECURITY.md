# Security Policy

## Reporting a security concern

If you discover sensitive information in this repository (credentials, private IPs, internal diagrams, unredacted incident details), please **do not** open a public issue.

Instead, contact the maintainer via GitHub (preferred):
- Open a private report if available, or
- Use the repository contact listed on the GitHub profile.

Include:
- A link to the file/location
- What you found
- Why it is sensitive

We will remove or remediate as quickly as practical.
